# Rotation Crop Farming Joint Venture Landing Page

A professional landing page designed to attract potential joint venture partners for a sustainable rotation crop farming business. This page showcases farming practices, partnership benefits, and provides contact information for interested investors and collaborators.

## Overview

This landing page presents our rotation crop farming joint venture opportunity to prospective partners. The site features information about our sustainable farming practices, the benefits of partnership, testimonials from existing partners, and a contact form for interested parties.

## Technology Stack

- **HTML5** - Semantic markup for structure and accessibility
- **CSS3** - Modern styling with custom properties, Grid, and Flexbox
- **Vanilla JavaScript** - Interactive features and form handling

## Local Development

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Python 3.x (for local server) or any other HTTP server

### Setup Instructions

1. Clone or download this repository to your local machine

2. Navigate to the project directory:
   ```bash
   cd rotation-crop-farming-joint-venture-landing-page
   ```

3. Start a local development server using Python:
   ```bash
   python -m http.server 8000
   ```

   Or using Python 3:
   ```bash
   python3 -m http.server 8000
   ```

4. Open your web browser and navigate to:
   ```
   http://localhost:8000
   ```

### Alternative Server Options

If you don't have Python installed, you can use other HTTP servers:

- **Node.js http-server**:
  ```bash
  npx http-server -p 8000
  ```

- **PHP built-in server**:
  ```bash
  php -S localhost:8000
  ```

- **VS Code Live Server extension**: Right-click on `index.html` and select "Open with Live Server"

## Deployment

### GitHub Pages

1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Select the branch to deploy (usually `main`)
4. Set the folder to `/ (root)`
5. Click Save

Your site will be published at: `https://[your-username].github.io/[repository-name]`

### Netlify

1. Create a Netlify account at [netlify.com](https://www.netlify.com)
2. Drag and drop the project folder to Netlify's deploy area
3. Your site will be live instantly with a generated URL
4. Optionally, configure a custom domain in site settings

### Vercel

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project directory
3. Follow the prompts to deploy
4. Your site will be live with a generated URL
